# Practices Skills Index

This directory contains practice-oriented skills focused on planning, slicing, refactoring, and simplifying software work.

## Available Skills

| Skill | What it is for |
| --- | --- |
| [`code-simplifier`](./code-simplifier/SKILL.md) | Simplifies and refines code for clarity, readability, and maintainability without changing behavior. |
| [`complexity-review`](./complexity-review/SKILL.md) | Reviews technical proposals across complexity dimensions and challenges unnecessary scale or architectural weight. |
| [`hamburger-method`](./hamburger-method/SKILL.md) | Breaks large features into vertical, end-to-end slices using the Hamburger Method. |
| [`refactoring`](./refactoring/SKILL.md) | Provides a structured refactoring workflow for improving messy or overly complex production code. |
| [`refinement-loop`](./refinement-loop/SKILL.md) | Guides iterative, multi-pass refinement when a single draft is not enough. |
| [`small-safe-steps`](./small-safe-steps/SKILL.md) | Plans implementation and migration work in 1–3 hour, low-risk increments. |
| [`story-splitting`](./story-splitting/SKILL.md) | Detects oversized stories and applies practical heuristics to split them into smaller deliverables. |
| [`thin-wrappers`](./thin-wrappers/SKILL.md) | Encapsulates infrastructure SDKs behind narrow, domain-aligned interfaces. |
| [`thinkies`](./thinkies/SKILL.md) | Applies Kent Beck’s Thinkies to generate alternatives and reframe decisions when stuck. |

## Notes

- Each skill lives in its own folder.
- The primary entry point for each one is its `SKILL.md` file.
